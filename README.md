# llmsec.dev — LLMSecTest website

Source for the project website at **https://llmsec.dev**.

LLMSecTest is a pytest-native security testing framework for LLM applications,
mapped to the [OWASP LLM Top 10](https://owasp.org/www-project-top-10-for-large-language-model-applications/).
Code lives at [github.com/wehnsdaefflae/llmsectest](https://github.com/wehnsdaefflae/llmsectest).

Funded by the [Prototype Fund](https://prototypefund.de) (FKZ 16IS26S10). MIT-licensed.
